\subsection{Les systèmes d'archivage}

\frame{\frametitle{L'archivage classique sur supports physiques}
Supports typiques : cylindres de cires, bandes magnétiques analogiques, disques vinyles, DAT, CD, etc...

\begin{columns}[c]

\column{6cm}
\begin{itemize}
 \item Buts :
    \begin{itemize}
    \item Conformité à l'oeuvre originale
    \item Reproductibilité
    \item Accessibilité
    \end{itemize}
 \item Problèmes :
    \begin{itemize}
    \item Fragilité : rayures, fissures, changement de phase du matériau, syndrôme du vinaigre,...
    \item Nécessité d'appareils de lecture specifiques
    \item Pérennité
    \end{itemize}
\end{itemize}

\subsection{Exemple : Le fond d'archives du CREM}
\frame{\frametitle{Le fond d'archives du CREM}
Un fond ethnomusicologique unique :
\begin{itemize}
 \item 4291 collections (inédit et édité)
 \item Actuellement 22101 fiches documentaires comportant 50 champs en moyenne (60000 environ à terme)
 \item 3250 heures de sons inédits soit 4 To environ + montages et copies diverses
 \item 3500 heures de sons édités soit 4,5 To environ
 \item 300 heures environ de video ($\approx$ 4 To)
\end{itemize}
\vspace{0.5cm}
Besoins en numérisation, en traitement documentaire (reste 65\% du fond à effectuer) et en \textbf{valorisation} ! \\

\subsection{Objectifs du projet Telemeta}
\frame{\frametitle{Objectifs du projet Telemeta}
\begin{itemize}
 \item Pérenniser les archives audionumériques
 \item Valoriser le patrimoine culturel par la consultation légale
 \item optimiser la transmission des méta-données
 \item Augmenter les capacités de recherche (web sémantique, interopérabilité, croisement de données)
 \item Définir un systématisme de sauvegarde des oeuvres audiovisuelles
\end{itemize}

\subsection{Technologies}
\frame{\frametitle{Les technologies utilisées par Telemeta}
Open Source :
\begin{itemize}
  \item Licence CeCILL
  \item Pérennise les ressources
  \item Dynamise le développement (partage, communautés internationales)
  \item Limiter les coûts à grande échelle

Briques :
\begin{itemize}
 \item Python : langage\\
    \url{http://python.org}
 \item Django : framework \\
    \url{http://djangoproject.com}
 \item MySQL : base de données relationnelle\\
    \url{http://mysql.com}
 \item Linux, OSX, Windows : système multi-plateforme
\end{itemize}
}

\subsection{Standards}
\frame{\frametitle{Les standards et normes utilisés par Telemeta}
\begin{itemize}
 \item Web :
    \begin{itemize}
     \item HTML : langage hypertextuel
     \item CSS : styles
     \item SQL : base de données
    \end{itemize}
 \item Audio :
    \begin{itemize}
    \item WAV : archivage audio brut
    \item MP3, MP4 : compression avec pertes, largement utilisé, encapsulation partielle
    \item OGG Vorbis : compression avec pertes, open source, encapsulation totale
    \item FLAC : compression sans pertes, multi-pistes, open source, encapsulation totale
    \end{itemize}
 \item Métadonnées :
    \begin{itemize}
     \item XML (W3C)
     \item DublinCore (OAI-PMH)
     \item exports SQL
     \item (OWL : Web Ontology Language)
    \end{itemize}
\end{itemize}
}
